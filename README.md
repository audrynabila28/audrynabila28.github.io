# 🌐 Personal Website Portfolio - Audry Nabila

Website portofolio pribadi modern, responsif, dan estetis yang dirancang dengan nuansa gelap (*dark theme*) dan aksen gradasi neon teal/indigo. Halaman ini berfungsi sebagai kartu nama digital dan portofolio interaktif untuk memamerkan keahlian di bidang **Backend Engineering** dan **Data Analysis**.

---

## 🚀 Fitur Utama Website
1.  **Typewriter Animation**: Efek ketik otomatis pada bagian sub-judul hero section untuk memperlihatkan keahlian inti.
2.  **Interactive Stats Card**: Menampilkan statistik singkat proyek dan tingkat akurasi analisis data.
3.  **Visual Tech Stack Badges**: Lencana kategori keahlian (Backend, Data Analyst, Tools) yang interaktif dengan micro-hover scale effect.
4.  **Showcase Proyek**: Kartu khusus untuk proyek **Humaira Salon** (disertai link repositori & web aktif) dan **Asahlagi Capstone Project**.
5.  **WhatsApp Direct Chat**: Form kontak interaktif yang langsung mengarahkan pengunjung ke obrolan WhatsApp pribadi Anda.

---

## 🛠️ Tech Stack Halaman Web
*   **Structure**: HTML5
*   **Styling**: Custom CSS3 (Custom grid, gradients, dan glassmorphism)
*   **Logic**: Vanilla JavaScript ES6+ (Typewriter engine, smooth scrolling scrollspy, & menu toggle mobile)
*   **Fonts & Icons**: Google Fonts (Outfit & Plus Jakarta Sans), FontAwesome Icons

## 📂 Cara Deploy ke GitHub Pages (Menjadikannya Online Gratis!)

Agar portofolio ini dapat diakses publik dengan alamat web **`https://audrynabila28.github.io`**, ikuti langkah mudah berikut:

1.  **Buat Repositori Baru di GitHub**:
    *   Buka GitHub dan klik tombol **New Repository**.
    *   Beri nama repositori tersebut **HARUS persis** seperti ini: **`audrynabila28.github.io`** (menggunakan username Anda).
    *   Pilih opsi **Public**.
    *   *Jangan* centang "Add a README file" (karena kita sudah punya README ini).
2.  **Upload File Portofolio**:
    *   Di halaman repositori kosong tersebut, klik **uploading an existing file**.
    *   Drag dan drop semua file dari folder `Personal-Portfolio` ini:
        *   `index.html`
        *   `style.css`
        *   `script.js`
        *   `README.md` (file ini)
    *   Klik **Commit changes** di bagian bawah.
3.  **Selesai!**:
    *   GitHub Pages akan memproses website Anda dalam waktu kurang dari 1 menit.
    *   Buka peramban (*browser*) dan akses: **`https://audrynabila28.github.io`** 🚀

## ⚡ Alternatif: Deploy via Vercel
1.  Upload folder ini ke repositori GitHub biasa dengan nama bebas (misal: `my-portfolio`).
2.  Masuk ke dashboard **Vercel.com**, klik **Add New** -> **Project**.
3.  Import repositori `my-portfolio` tersebut.
4.  Klik **Deploy** (tanpa mengubah konfigurasi apa pun karena ini adalah website statis murni). Anda akan mendapatkan alamat web gratis aktif dari Vercel!
